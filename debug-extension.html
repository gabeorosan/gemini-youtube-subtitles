üö® Common Issues & Solutions

Issue 1: Raw SRT Text Displayed

Symptoms: Seeing text like "1\n00:00:00,000 --> 00:00:03,000\nSubtitle text"

Cause: SRT parsing failed, fallback returned raw text

Solution: Check console for parsing errors, verify Gemini output format

Issue 2: No Formatting Applied

Symptoms: Text appears but without styling/structure

Cause: CSS not loading or HTML structure incorrect

Solution: Check if styles.css is loaded in manifest.json

Issue 3: String Instead of Array

Symptoms: Console shows "dataType: string" instead of "object"

Cause: Background script returning raw text instead of parsed objects

Solution: Check background script parsing logic

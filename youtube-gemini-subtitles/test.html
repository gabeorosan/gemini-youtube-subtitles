Chrome Extension Test

Open this page and load the extension to test it.

Instructions:

  1. Open Chrome and go to chrome://extensions/
  2. Enable "Developer mode"
  3. Click "Load unpacked" and select the youtube-gemini-subtitles folder
  4. Go to a YouTube video page
  5. Use Cmd+Shift+P (Mac) or Ctrl+Shift+P (Windows) to test
  6. Check the browser console for debugging information
